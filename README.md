# FocusTube

FocusTube is a Chrome extension designed to help students stay focused on YouTube by allowing videos only from selected educational channels.

## Features
- Whitelist YouTube channels
- Hide distracting videos
- Simple popup UI
- Works with dynamically loaded YouTube content

## How to Install
1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder

## Tech Stack
- JavaScript
- Chrome Extensions (Manifest V3)
- HTML, CSS
